# Notices

This project may reference or transform third-party open-source material during
benchmarking and instruction-data generation.

## Third-Party Sources Referenced by Local Experiments

- `Harrison1/unreal5-code-examples` was used locally as a permissive Unreal
  Engine example source. The repository was identified as MIT licensed during
  local preparation.
- GitHub issue benchmark prompts include repository and issue metadata from
  permissive-license public repositories. The benchmark keeps source URLs so
  provenance remains visible.

## Not Redistributed by Default

The public repository should not include:

- downloaded repository archives,
- extracted source corpora,
- tokenized datasets,
- model checkpoints,
- private project scripts,
- copied Unreal/Epic documentation corpora.

If model weights or datasets are published later, include a separate notice
file listing the exact source repositories, versions, licenses, and URLs.
